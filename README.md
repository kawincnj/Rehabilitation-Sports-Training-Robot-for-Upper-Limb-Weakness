# 🤖 Rehabilitation Sports Training Robot for Upper Limb Weakness  
### 🏅 SIIF 2025 — **Gold Medal**

A **smart rehabilitation robot system** designed to support patients with **upper limb muscle weakness**, enabling **remote, home-based rehabilitation** while maintaining professional-level training quality through AI-assisted motion analysis.

---

## 🩺 Problem Statement
Patients with upper limb weakness often need to travel to hospitals **weekly** for rehabilitation:
- ⏱️ Time-consuming travel
- 💸 High transportation costs
- 😞 Reduced motivation and fatigue from repetitive therapy

These barriers limit long-term rehabilitation effectiveness.

---

## 💡 Our Solution
This project introduces a **Rehabilitation Sports Training Robot** that allows patients to:
- Train at home  
- Perform guided arm movements  
- Receive motion analysis and progress tracking via a **web application**

The system combines **robotics, IoT, and web technology** to deliver accessible and engaging rehabilitation.

---

## 🏗️ System Overview
**Hardware + Software Integrated System**

### Hardware
- Upper-limb assistive robotic arm
- Servo motors for controlled movement
- Arduino nano ESP32 microcontroller
- Safety-focused mechanical design

### Software
- Web application built with **React**
- Backend communication via **MQTT**
- Real-time motion data monitoring
- Training session visualization

---

## 🚶 User Journey
1. User logs into the web application  
2. Rehabilitation device is connected  
3. User performs guided arm exercises  
4. Robot assists movement when needed  
5. System records motion data  
6. Progress is visualized on the dashboard
